AI Backend

Когнитивный AI-ассистент с долгосрочной памятью, многоагентной архитектурой и learning spine. Система не просто отвечает — она строит модель пользователя, отслеживает цели, инициирует разговор и обучается на обратной связи.

  • Goals / Tasks / Habits система с API

  • Learning spine: поведенческие политики из обратной связи
  • Голосовые сообщения (Groq Whisper)
